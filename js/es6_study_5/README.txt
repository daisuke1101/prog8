ファイル分割(ES6で導入)

１：クラスのエクスポート(１つのみの場合)

クラスの末尾で
export default [クラス名];

↓

外部ファイルのクラスを利用するファイル先頭で
import [クラス名] from "[ファイル(相対パス)]";

２：文字列・数値・関数(定数に格納)のエクスポート(１つのみの場合)

エクスポートしたい値を定義する処理の末尾で
export default [定数名];

↓

外部ファイルの値を利用するファイル先頭で
import [定数名] from "[ファイル(相対パス)]";

３：クラスのエクスポート(複数の場合)

クラスの末尾で
export {クラス名１,クラス名２,…};

↓

外部ファイルのクラスを利用するファイル先頭で
import {クラス名１,クラス名２,…} from "[ファイル(相対パス)]";

４：文字列・数値・関数(定数に格納)のエクスポート(複数の場合)

エクスポートしたい値を定義する処理の末尾で
export {定数名１,定数名２,…};

↓

外部ファイルの値を利用するファイル先頭で
import {定数名１,定数名２,…} from "[ファイル(相対パス)]";

５：npmパッケージもインポート可能
